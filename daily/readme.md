/daily/ ← 存放每日会话内容的文件夹  
……/2026-03-16/  
…………session-notes.md ← 当天会话内容  
……/2026-03-17/

## 2️⃣ 各 md 文件规则和内容

### 2.1 `ai_guide.md`（AI 辅导老师使用指南）

内容示例（针对 Python 学习）：

# AI 辅导老师使用指南 - Python 学习

## 角色

- 你是一个耐心的 Python 学习导师
- 使用苏格拉底式教学法，引导学生探索和发现
- 不直接给出答案，而是通过提问和示例帮助理解

## 哲学与方法

1. **先问基础**：了解学生已有知识
2. **循序渐进**：分步骤讲解复杂概念
3. **理解检查**：通过提问或小练习验证掌握情况
4. **适应性跟进**：根据学生理解调整讲解方式

## 内容范围（Python 学习示例）

- 基础语法（变量、数据类型、运算符）
- 控制流（条件语句、循环）
- 函数与模块
- 数据结构（列表、字典、集合、元组）
- 面向对象编程
- 文件操作
- 异常处理
- 常用标准库（datetime、math、os 等）
- 小项目实践（计算器、文本分析、小游戏等）

## 使用流程

1. 学生提出问题或回答练习
2. AI 根据问题先了解基础，再提供讲解与示例
3. 学生完成理解检查，AI 给出反馈
4. 学生或 AI 记录每日会话到 `daily/YYYY-MM-DD/session-notes.md`
5. 更新 `progress.md` 反映新掌握知识点和待补习内容

## 注意事项

- 每次学习前，AI 可查看最新的 `ai_guide.md` 和 `progress.md` 了解学生状态
- 不修改 `session-template.md`，仅作为每日会话记录模板

内容示例：

# Python 学习进度总览

## 已掌握主题

| 主题           | 掌握程度 | 学习日期   | 备注                     |
| -------------- | -------- | ---------- | ------------------------ |
| 变量与数据类型 | 高       | 2026-03-15 | 理解基本类型和赋值       |
| 条件语句       | 中       | 2026-03-16 | 理解 if/else，需更多练习 |

## 知识盲点

| 主题     | 严重程度 | 备注                 |
| -------- | -------- | -------------------- |
| 面向对象 | 高       | 类和对象理解不够深入 |
| 文件操作 | 中       | 读写文件语法掌握不足 |

## 行动计划

- 复习面向对象和文件操作
- 完成小项目练习（例如文本分析器）
- 下一次学习目标：函数和模块

> 每次学习结束后，你手动或通过 AI 更新这个文件的内容，保持学习总览完整。

## 3️⃣ 网站展示逻辑

- **学习日报页面 `daily.html`**：
  
  - 展示 `daily` 文件夹中所有每日会话内容，按日期排序
    
  - 列表顶部固定显示 `ai_guide.md` 和 `progress.md`和`readme.md`和`session-template.md`和`python-learning-path.md`（可以通过人为设置日期或者增加标记排序）
    
  - 点击文件可查看详情页面
    
- **新增每日会话**：
  
  1. 在 `/daily/YYYY-MM-DD/` 创建新文件夹
    
  2. 学习完成后用ai使用 `session-template.md` 生成 `session-notes.md`
    
  3. 更新 `progress.md`，保持总览最新
    
  4. `ai_guide.md` 可在学习前参考或按需要更新
    

---

## ✅ 整体流程总结

1. **学习前**：
  
  - AI 查看 `ai_guide.md` + `progress.md` 了解学习目标与状态
2. **学习中**：
  
  - 使用 AI 对话进行提问、讲解和理解检查
3. **学习后**：
  
  - 将当天会话记录在 `/daily/YYYY-MM-DD/session-notes.md`
  - 更新 `progress.md` 反映新掌握知识
    
4. **网站展示**：
  
  - 学习日报页面列表展示所有 md 文件（每日会话 + 指南 + 进度） 
  - 点击查看详情，按日期排序，`ai_guide.md` 和 `progress.md`和`readme.md`和`session-template.md`和`python-learning-path.md` 固定在列表顶部
