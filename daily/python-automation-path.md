# Python 自动化测试学习路线

## 阶段 0：学习准备（1-2天）
* 安装 Python、VS Code / PyCharm
* 明确目标：自动化测试为主
* 熟悉命令行、Git 基础

---

## 阶段 1：Python 基础（15天）
* 变量与数据类型（int, float, str, bool）
* 运算符、表达式
* 条件语句（if / elif / else）
* 循环语句（for / while）
* 字符串操作、列表、字典、元组、集合
* 函数基础、参数、作用域、lambda
* 面向对象：类、对象、方法、封装
* 模块与文件操作
**目的**：打好脚本基础，能处理数据和逻辑
**里程碑**：编写一个处理 CSV/JSON 数据的自动化登录校验脚本

---

## 阶段 2：Python 进阶（10天）
* 异常处理（try / except / finally，自定义异常）
* 文件操作（读写 CSV、Excel、JSON）
* 常用标准库：os, sys, datetime, json, logging
* 函数进阶：闭包、装饰器、高阶函数
* 并发基础（多线程、多进程、协程 async/await）
**目的**：掌握脚本高级能力，能应对复杂自动化任务
**里程碑**：编写一个带自动日志记录与失败重试机制的工具库

---

## 阶段 3：自动化测试实战（15-20天）
* Python 测试框架：pytest、unittest
* Web 自动化：selenium / playwright
* 接口自动化：requests + json / pytest
* 数据驱动测试（Excel / CSV 数据输入）
* 日志、报告生成
* CI/CD 入门：Jenkins 或 GitLab CI
* 实战项目：接口测试脚本 + Web 测试脚本
**目的**：能独立完成公司级自动化测试任务
**里程碑**：搭建一个包含 Web 与接口测试的完整 CI/CD 自动化框架

---

## 阶段 4：附加方向（可选）
* Web 开发（Flask / FastAPI / Django）
* 爬虫开发（requests / BeautifulSoup / Scrapy）
* 数据分析（Pandas / Matplotlib / Seaborn）
* AI 开发（TensorFlow / PyTorch / LangChain）

---

## 阶段 5：项目整合与提升
* 自动化测试 + 数据分析 + 脚本工具整合
* 建立个人测试项目合集
* 模拟企业 CI/CD 环境
* 持续优化代码规范（PEP8 / Pythonic）
