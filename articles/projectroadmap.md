# 全栈学习+项目实践方案

## 一、学习目标与知识模块

| 层级  | 技能/模块 | 学习目标 | 对应网站功能 |
| --- | --- | --- | --- |
| 前端基础 | HTML、CSS、JS | 页面结构、样式和交互 | 首页布局、文章/视频展示、搜索、分页 |
| 前端框架 | Vue / React | 组件化开发、数据绑定、响应式 | 可以用 Vue 改造文章列表、视频列表，组件化管理卡片 |
| 后端基础 | Python+Flask/ Node.js + Express | 服务器逻辑、API 处理 | 文章、视频 JSON 数据 API，数据上传接口 |
| 数据库 | MySQL / PostgreSQL / MongoDB | 数据存储、查询、关联 | 存储文章、视频、标签、分类，提供 API 查询 |
| API 调用 | RESTful / GraphQL | 与前端或第三方交互 | 后端接口提供分页、搜索、分类等数据 |
| 部署运维 | Linux / Docker / Kubernetes | 部署项目、服务化管理 | 将网站打包、容器化，部署到本地或云端，Nginx 反向代理 |
| 测试与优化 | 单元测试 / 集成测试 / 性能优化 | 保证系统稳定 | JS 单元测试、API 测试、页面渲染性能优化 |

## 二、网站结构设计

- **首页 (index.html)**：
  
  - 展示最新文章和视频，限制6条
    
  - 支持点击视频和文章查看
        
- **文章列表页 (articles.html)**：
  
  - 支持文章搜索功能
    
  - 分页展示所有文章内容，每页6个
    
  - 支持点击文章标题查看
    
  - <mark>文章摘要+标签展示</mark>
    
  - <mark>提供各类学习内容页签，如前端，python，linux等，与文章内容标签关联</mark>
    
  - <mark>分类筛选：文章内容可按照标签分类</mark>
    
- **视频列表页 (videos.html)**：
  
  - 支持视频搜索功能
    
  - 分页展示所有视频内容，每页6条
    
  - 点击视频卡片播放
    
- **单文章页 (article.html)**：
  
  - 渲染markdown，支持嵌入图片/表格/图表/代码
    
  - 代码高亮、折叠、复制
    
- **单视频页 (video.html)**：
  
  - 视频播放
    
  - 展示视频名称和简介
    
- **关于页面 (about.html)**：
  
  - 网站描述

## 四 、学习网站实施步骤（短期 → 中期 → 长期）

### **短期（本地静态 + JSON）**

- 用 Markdown + JSON 存储文章和视频信息
  
- 前端 JS 渲染内容（分页、搜索、分类）
  
- 保持静态网页，学习前端布局和交互
  
- 学习如何用 JS 操作 DOM、分页、过滤
  

### **中期（本地 API + 简单后端）**

- 建立本地 Node.js / Python 后端
  
- 将 JSON 替换成 API 返回数据
  
- 文章/视频可通过后端上传，前端调用 API 渲染
  
- 学习 API 调用和请求处理
  

### **长期（全栈 + 部署）**

- 使用数据库存储文章和视频
  
- 前端框架改造网站（组件化、状态管理）
  
- 后端提供完整 REST API 或 GraphQL
  
- 部署到云服务器或 Docker + K8s 集群
  
- 未来可加登录系统、权限管理，逐步变成小型 CMS
  

## 五、网站功能与学习结合的设计理念

- **每篇文章或视频都对应一个技能点**，可用标签标记
  
- **首页显示最新学习进度**，形成“学习日志”
  
- **分类页 + 搜索功能**，方便你管理全栈学习内容
  
- **未来扩展 API 和数据库**，可以直接用现有 JS 渲染函数替换 JSON 数据源即可
